Project Name: Melodia

Description: 
Transcribes .WAV files (audio) into sheet music with Aubio and also allows the 
user to input their own single-note melodies into the image processing algorithm.

How To Run:
Download all the files in the zip folder and make sure you have access to the 
internet (the images for the music are online).


Libraries:
- Tkinter
- PIL(Pillow)
- Aubio

Shortcuts:
In startOptionsMode (after splashscreen):
- press "a" for samples mode
- press "s" for transcription mode
- press "d" for view mode
